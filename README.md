### NeuroFactory: Ontology-Grounded Hybrid RAG for Natural Language Querying in MES
*Capstone Design Project · Role: Frontend Developer*
- Built the React-based frontend UI for an on-premises NL-to-SQL system, letting factory engineers submit Korean/English natural-language queries over a live MES (THiRA MES 3.1.0) and view results without SQL knowledge.
- Implemented query input, tabular result visualization, validation-error feedback, and diagnostics views in the user interface.
- Integrated the frontend with the FastAPI backend (REST API), wiring the end-to-end query pipeline (retrieval → SQL generation → safety validation → execution) to the UI.
- Contributed to a system achieving **89.5% execution accuracy** and **100% valid-SQL rate** on the MES-NL2SQL-Gold benchmark (n=200), a **+52.5 pp** gain over the schema-only baseline.
